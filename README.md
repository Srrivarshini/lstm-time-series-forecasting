# Advanced Time Series Forecasting with LSTM

This project implements a multivariate time series forecasting solution using Long Short-Term Memory (LSTM) neural networks.

## Features
- Synthetic multivariate dataset with trend and seasonality
- Time-aware preprocessing and sliding window sequence generation
- LSTM-based forecasting model
- Evaluation using RMSE, MAE, and MAPE
- Explainability using permutation feature importance

## Tech Stack
- Python
- NumPy, Pandas
- Scikit-learn
- TensorFlow (Keras)
- Matplotlib

## Objective
To accurately forecast future energy consumption while providing interpretability into feature contributions.
